# 👶 Childbirth Mode Prediction System

An end-to-end explainable AI system for predicting childbirth delivery modes with hospital-grade web interface.

## 🚀 Features

### 🤖 AI/ML Capabilities
- Ensemble machine learning model (XGBoost, Random Forest, KNN)
- SHAP-based explainable AI for clinical interpretability
- Real-time prediction with confidence scores
- Risk score calculation based on clinical factors

### 🏥 Clinical Features
- Professional hospital-grade interface
- Secure login authentication
- Dynamic patient input forms with validation
- Real-time BMI calculation
- Clinical recommendations based on predictions
- High-risk case alert system

### 📊 Analytics & Reporting
- Interactive analytics dashboard
- Patient history management
- XAI (SHAP) explanation explorer
- Automated PDF report generation
- Delivery mode trend analysis
- Risk distribution visualization

### 💻 Technical Features
- Single-file Streamlit application
- Modular code structure
- CSV-based patient storage (no database required)
- Complete error handling
- Responsive design
